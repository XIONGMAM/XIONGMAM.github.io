二、 GitHub 管理

  • 官网下载VSCode 并安装
  • 打开VSCode, 在左侧工具栏点开 源代码管理 => 安装Git 64位
  • 用windows PowerShell 输入 git 如果有 输出就可以正常使用，如果报错，百度搜解决方法
  • 打开VSCode ，新建项目 => 写好代码 => 点开源代码管理 => 初始化存储库 （所有文件都被提交暂存库）
  • 点击源代码管理 旁边的 对勾 提交 => 输入 master => 三小点选 推送到 => 添加远程
  • 从 GitHub 添加 => 登录GitHub => 登录后会获取自己的仓库 => 此时不要选仓库，而是先创建仓库，防止污染
  • 到GitHub上新建仓库 => 然后返回vscode => 点 推送到 => 添加远程 => 从GitHub
  • 输入仓库名称 => 输入远程库的名称 master
  • 返回VSCode => 源代码管理点三个点 => 推送 => 添加远程 => 输入刚新建的仓库名称（注意大小写）
  • 推送上去就可以了
  • .gitignore 新建一个忽略文件， 选中要忽略的文件右键，复制相对路径，最前面加/ ，其余的 \ 改成 /
  • 右键run code 想在终端输出，可以到设置，输入 run in terminal 勾选选项
  • 放大代码 设置 => 输入 zoom 勾选参数，这样就可以按住ctrl 放大代码
  • 流程
    1. 代码(正本) => clone (副本) => add (添加代码) => commit(提交) => push(添加到正本) => pull(拉取最新代码)
    2. diff代码做对比 <= fetch先更新到本地版本库 <= pull工作区 <= push远程仓库 <= commit本地仓库 <= add暂存区 <= 工作区
    3. Stage(已追踪) Untrached(未知) unstage(未追踪)
    4. master(版本主支) => 版本2新建分支
  • 实操
    • 命令描述
      git config --global user.name "XIONGBAB"设置全局名称
      git config --global user.email 2770949518@qq.com设置全局邮箱
      git init初始化
      echo "版本1" > Markdown.md创建MarkDown文档
      git status查看状态
      git add Markdown.md添加到暂存区
      git commit进入vim 进行编辑，输入内容后保存退出编辑模式 # 后面的说明是不会显示出来，保存退出
      git commit -m ""不进入vim编辑，直接在引号里添加提交信息
      git checkout master切换到master
      git branch -d name删除分支，因为没有合并分支，且要删除分支，直接用 -D
      git checkout -b name创建并马上切换到新分支，用-b
      git commit -am ""提交后，进行下一步合并的操作，可以合并在一起写
      git checkout master进入主分支
      git merge name把别的分支合并到当前所处的分支上，从其他地方合并过来。如果分支要从主支合并过来，手动保留
      git removte -v查看本地仓库和哪些远程仓库有联系
      git push回车后输入用户名和密码，21年后不支持输入密码，需要使用token
      git push回车后输入用户名，再粘贴新创建的 token ，粘贴一次就可以了，默认不显示，直接回车
      git fetch拉取文件到本地版本库
      git diff origin/main加上远程仓库名和分支名，就查看本地和远程仓库的区别，当没有问题时，再执行拉取
      git pull把远程库整合到工作区
      git log就可以查看所有版本历史

三、快速生成HTML 结构语法

  • 生成标签直接输入标签名按tab键即可比如 div 然后 tab 键，就可以生成 <div></div>
  • 想要生成多个相同标签加上*就可以了，比如 div*3 就可以快速生成 3个div
  • 有父子级关系的标签，可以用>比如 ul > li 就可以了
  • 有兄弟关系的标签，用+就可以了比如 div + p
  • 生成带有类名或者id名字的，直接写 .demo或者 #two tab键就可以了
  • 生成的div类名是有顺序的，可以用自增符号$
  • 输入Lorem10 Lorem后面跟数量， 就会随机出现10个单词，节省输入单词时间
  • 输入div{$@7}*3 意思是从序号为7，开始创建3个div

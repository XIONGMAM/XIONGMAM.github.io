一、HTML5新增的语义化标签

  • 标签
      <header>            <!-- 头部标签 -->
  <nav>               <!-- 导航标签 -->
  <article>           <!-- 内容标签 -->
  <section>           <!-- 定义文档某个区域 -->
  <aside>             <!-- 侧边栏标签 -->
  <footer>            <!-- 尾部标签 -->
  • 注意：
    1. 这种语义化标准主要是针对搜索引擎的
    2. 这些新标签页面中可以使用多次
    3. 在IE9中,需要把这些元素转换为块级元素
    4. 其实,我们移动端更喜欢使用这些标签
    5. HTML5还增加了很多其他标签,我们后面再慢慢学

二、HTML5新增的多媒体标签

  • 简介：
    1. 使用它们可以很方便的在页面中嵌入音频和视频,而不再去使用flash和其他浏览器插件
    2. HTML5在不使用插件的情况下,也可以原生的支持音频格式文件的播放,当然,支持的格式是有限的。
    3. 谷歌浏览器把音频和视频自动播放禁止了
  • 音频标签：<audio>
    1. 语法: 当前<audio元素支持三种音频格式： WAV Ogg MP3 (IE Safari 不支持Ogg , IE 不支持WAV)
      • Html 
          <!-- 语法1： -->
  <audio> src="./videos/v.mp3" controls="controls"></audio>
  
  <!-- 语法2： -->
  <audio controls="controls">
      <source src="music.ogg" type="audio/ogg">
      <source src="musci.mp3" type="audio/mpeg">
      您的浏览器暂不支持>audio><标签   <!-- 这段如果支撑是不会显示,不支持则会出现 -->
  </audio>
    2. 属性
      • 属性值描述
        autoplayautoplay如果出现该属性,则音频在就绪后马上播放
        controlscontrols如果出现该属性,则向用户显示控件,比如播放按钮
        looploop如果出现该属性,音频结束时会重复播放
        srcurl音频url地址
  • 视频标签：<video>
    1. 语法: 当前<video>元素支持三种视频格式：WebM Ogg MP4 (尽量使用mp4格式,其它考虑兼容性)
      • Html 
          <!-- 语法1： -->
  <video src="./videos/v.mp4" controls="controls"></video>
  
  <!-- 语法2： -->
  <video width="300" controls="controls">
      <source src="move.ogg" type="video/ogg">
      <source src="move.mp4" type="video/mp4">
      您的浏览器暂不支持>video><视频标签   <!-- 这段如果支撑是不会显示,不支持则会出现 -->
  </video>
    2. 属性
      • 属性值描述
        autoplayautoplay视频就绪自动播放(谷歌浏览器需要添加muted来解决自动播放问题)
        controlscontrols向用户显示播放控件
        widthpixels(像素)设置播放器宽度
        heightpixels(像素)设置播放器高度
        looploop播放完是否继续插放该视频,循环播放
        preloadauto/none规定是否预加载视频(如果有了autoplay 就忽略该属性,none为不加载)
        srcurl视频url地址
        posterimgurl加载等待的画面图片
        mutedmuted静音插放,部分浏览器需要加上此属性
  • 配合js
    • Javascript 
        <audio src="./sound/game_music.mp3" id="startMusic"></audio>  // 路径
  <audio src="./sound/bullet.mp3" id="bulletMusic"></audio>     // 路径
  
  let startMusic = document.getElementById("startMusic");       // 获取音频元素
  let bulletMusic = document.getElementById("bulletMusic");     // 获取元素
  startMusic.play();                                            // 调用音频播放方法
  bulletMusic.play();                                           // 调用音频播放方法
  startMusic.pause();                                           // 调用音频暂停播放方法

  /* audio不仅是一个标签,也是window下的一个对象,作为对象,具有一些对象属性和对象方法：
  对象属性：
      currentTime：获取当前播放时间
      duration：获取歌曲的总时间
      pause：是否暂停,返回布尔值
  对象方法：
      play()：播放
      pause()：暂停
      load()：重新加载 */

三、HTML5新增的 input 类型

  • 标签：我们验证的时候必须添加form表单域
    • Html 
        <li>邮箱： <input type="email"></li>          <!-- 限制用户输入必须为 Email 类型 -->
  <li>网址：<input type="url"></li>             <!-- 限制用户输入必须为 URL 类型 -->
  <li>日期：<input type="date"></li>            <!-- 限制用户输入必须为 日期 类型 -->
  <li>月：<input type="month"></li>             <!-- 限制用户输入必须为 月 类型 -->
  <li>周：<input type="week"></li>              <!-- 限制用户输入必须为 周 类型 -->
  <li>时间：<input type="time"></li>            <!-- 限制用户输入必须为 时间 类型 -->
  <li>数量：<input type="number"></li>          <!-- 限制用户输入必须为 数字 类型 -->
  <li>手机号码：<input type="tel"></li>          <!-- 手机号码 -->
  <li>搜索：<input type="search"></li>          <!-- 搜索框 -->
  <li>颜色：<input type="color"></li>           <!-- 生成颜色选择表单-->
  <li><input type="submit" value="提交"></li>   <!-- 提交-->
  • 属性
    • 属性值描述
      requiredrequired表单拥有该属性表示其内容不能为空,必填
      placeholderplaceholder表单的提示信息,存在默认值将不显示
      autofocusautofocus自动聚焦属性,页面加载完成自动聚焦到指定表单
      autocompleteoff/on当用户在字段开始键入时,浏览器基于之前键入过的值,应该显示出在字段中填写的选项。 , 默认已经打开,如autocomplete="on",关闭 autocomplete ='of”,需要放在表单内,同时加上name属性,同时成功提交
      multiplemultiple可以多选文件提交
  •   input::placeholder{color: pink;}    /* 可以通过伪类修改提示信息的字体颜色 */ 
  outline: none;                      /* 去除鼠标点后的轮廓线 */

一、注册

  • 百度搜索注册 进入官网 https://github.com （左边是登录 右边是注册）
  • 注意：
    1. 因为github在国外服务器所以访问较慢或者无法访问，需要翻墙（Shadowsocks），用github搜软件
    2. 注意：私有仓库只能自己或者指定的朋友才有权限操作（收费）
    3. 如果邮箱没有收到验证右键，请登录邮箱设置白名单，才可以验证邮箱
    4. 重新发送的操作如下：头像settings >> 左侧Email >> 点击resend >> 重新发送即可

二、基本概念

  • Repository： 仓库，用于存放项目代码，每个项目对应着一个仓库
  • Fork ： 复制克隆项目，该fork的项目是独立存在的
  • Star： 收藏项目，方便下次查看
  • Pull request ： 发起合并请求，基于fork，当你fork别人仓库的时候， 你发现或者可以完善代码时可以发起
  • Watch：关注项目，可以接收到项目更新提醒
  • Issue：事务卡片，发现代码bug，但目前没有成型代码，需要讨论时可用
  • 注意：私有仓库只能自己或者指定的朋友才有权限操作（且收费）
  • Github主页:账号创建成功或点击网址导航栏github图标都可进入github主页： 该页左侧主要显示用户动态以及关注用户或关注仓库的动态；右侧显示所有的git库
  • Github仓库:仓库主页主要显示项目的信息，如：项目代码，版本，收藏/关注/fork情况等
  • 个人主页 个人信息：头像，个人简介，关注我的人，我关注的人，我关注的git库， 我的开源项目，我贡献的开源项目等信息

四、仓库管理

  • 新建仓库文件 Add file >> Create new file
  • 书写代码：注意，文件名称需要些文件后缀名，主页必须是 index.html，描述尽量写详细，方便日后查看
  • 提交后自动跳转到仓库主页
  • 编辑/删除文件：代码仓库中，点文件名，进入文件页，进行编辑/删除；
  • 每次编辑/删除，可以做适当备注修改或者删除的内容，然后提交
  • 被删除的文件可以点击commits 来进行查看
  • 上传文件点击upload files，可以把文件拖进去，也可以直接,点击 choose your files ，传输完成后，添加信息提交
  • 搜索仓库文件，点击go to file 也可以用快捷键 T
  • 下载文件

五、对lssues的操作

  • 作用：发现代码BUG，但是目前没有成型代码，需要讨论时用；或者使用开源项目出现问题时使用
  • 点击new lessus
  • 填写标题和内容，可以添加修饰，加粗下划线等，然后最后点击提交按钮
  • 作者收到后，解决回复完，可以点击 close lessue 关闭，关闭后可以在lesses列表下切换状态

六、操作技巧

  • Watch(关注)： 关注后会显示该仓库的动态
  • 在头像上点击，可以去往很多功能的入口
  • fork 复制后仓库是独立的，在star旁边
  • pull requests :如果需要同步，则可以发起请求，等待作者审核合，也可以在这里看到请求
  • 注意
  • 如何为开源项目做出贡献：
    1. 新建issue ,提交使用问题或者建议或者想法
    2. Pull request 步骤：1.fork项目 2.修改自己仓库的项目代码 3.新建pull request 4.等待作者操作（合并）

七、项目查找方式

  • In:description + 搜索名称
  • 名称 + 语言 + 星星数
  • LICENSE：.mit 查找资源，如果公司项目，需要看下项目里的证书能不能使用

八、访问github慢修改hosts1

  • windows
    1. 打开hosts文件：C:\Windows\System32\drivers\etc\hosts
    2. 用记事本等添加,ip地址百度
    3. 刷新DNS缓存 ipconfig /flushdns
  • Mac
    1. 1.新建一个访达窗口，同时按住shift command G三个键，进入前往文件夹页面
    2. 2.在输入框内输入/etc/hosts
    3. 3.找到hosts文件夹
    4. 4.由于hosts文件夹不可编辑，所以复制一份hosts文件先保存到本地桌面
    5. 5.在新的hosts文件夹里输入百度查找到的代码

九、个人网站创建

  • 访问域名： https://github用户名.github.io
  • 搭建步骤：
    1. 创建个人站点 -> 新建仓库（注意：仓库名必须是用户名.github.io）
    2. 在仓库下新建index.html的文件,创建一个index.html，并提交即可
    3. 访问输入：https://"github用户名". github.io
    4. 注意： github pages只支持静态网页，仓库里面只能是.html文件

十、项目站点

  • 进入项目主页，点击settings
  • 在settings页面，点击[Launch automatic page generator]来自动生成主题页面
  • 新建站点基础信息设置，设置完后，拉到下面进行提交
  • 选择主题并提交，生成网页
  • 可以下载源代码
  • 这里可以进入网页代码文件页面进行代码修改

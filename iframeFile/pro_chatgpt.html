一、ChatGPT使用方法

  • 获取详细信息：
    1. 请具体告诉我xxx
    2. 请提供具体案例
    3. 情报的来源,出处,以及参考文献？
    4. 关于xx请详细说明
    5. 没写完的内容告诉它 继续
  • 寻求解决方案:
    1. 有比xxx更好的方法吗？
    2. 如何才能做得更好？
    3. 我现在能做的是什么？
  • 判断对错
    1. 关于xxx,我的想法是这样的,这样是对的吗？
    2. 做了xxx,会有用吗？
  • 请在xx字以内
    1. 以严肃风/幽默风/鲁迅风/xx风格
    2. 尽量简短
    3. 通俗易懂的回答
    4. 条列式回答
    5. 口语化回答

三、结合ChatGPT+Discord写网页,logo设计

  • Discord介绍
    1. 列服务器相当于教学楼,右侧频道相当于教室,点主页可以看到在线人数
    2. 点击左侧 + 图标创建自己的教室,然后按下面操作添加服务器
    3. INFO,中的Announcement 右上角人群图标,可以看到成员名单,就可以看到官方的额Midjourney,点击后添加服务器
    4. 刚创建的教室,点击授权,点击验证,看到成员列表里有官方机器人即可,然后就可以绘画了
    5. 点击教室里文字频道的 + 号键,输入abstract,相当于在电脑上新建文件夹一样,分组,这样在画不同的东西查阅方便
    6. 生成AI作品
    • 参数
      1. /stealth (意思是私密模式,只有开了订阅才有)
      2. /imagine(意思是想象,然后回车,出现光标,在框内输入)
      3. /imagine[prompt (意思是描述,用英文单词去描述),例如：a cute rabbit]
      4. /imagine[prompt a cute pig --v 4 (--v 4 是midjourney的特定指令,然后回车,第一次的话需要同意条款,点accept)]
      5. --ar 3:2 == --aspect 3:2 (意思是控制宽高比例,以及支持了16：9)
        (1:1 默认纵横比)
        (5:4 常见框架和打印比例)
        (3:2 印刷摄影中常见)
        (7:4 高清电视屏幕和智能手机屏幕)
      6. niji (niji模型,适用于绘出二次元模型)
      7. 使用的话要设置下： /settings => 选择niji Mode (这样就不用输入后缀,默认会带这个参数指令出图)
        1.给出三视图关键词：front,side ,back angles
        2.给出风格
        3.给出人物具体特征：人物的着装,表情等等
        4.总结描述词,记得用英文阐述
        • 属性描述
          --style 4b风格,默认是4b,另一种4a
          --stylize
          --chaos (0~100) == -c (0~100)用于控制出图的多样性,值越高就能得到不同的构图和意想不到的风格
          --seed (0~4294967295)不写会随机分配一个值,在图标上鼠标悬停,可在右侧有个笑脸 添加反应的按钮, 输入env 出现信封图标后点击,midjourney 就会发一条私信,告诉你他的seed值是多少 , 同一组描述词,出图的可能性有2的32次幂 种可能性空间,且描述词一样的前提下,参数调换顺序是不影响出图一致性
          hot :: dog权重分隔符,如果不用分隔符就会生成热狗,而不是很火热的狗狗
          hot ::1.5 dog 30:15:1 == 6:3:1可使用小数点,总和必须为正数不能为负数
          --no值可选 降权,默认为1,可以负面降权
          --tile无缝贴图
          --test --testp模型,v1 - v3 模型,v5模型
          --quality == -q .25/.5/1/2四种参数,数值越大不代表像素越大,画质越高,v4和niji不支持 --q 2,只影响初始的四宫格
          --upanime是niji模型默认用到的升档器,加上之后,点击U 按钮就会使用这个升档器
          --upbeta(升档的图片很光滑)
          --uplight(只添加少量的纹理)
    • 生成的图片下方参数说明(详细可以看官方帮助文档)
      u1-u4 v1-v4的意思：
      u代表升档,放大时会模糊,升档会提升细节和质量,v代表变体,微小的变化 1对应的第一张图标,对应4张,从上至下
      web按钮：点击后,可以右下角保存按钮进行图片保存
      upscale redo：不同的算法进行图片的放大
      循环按钮：重做的意思
      四个笑脸：评级功能
    • 到midjourney.com 官网 => sign in => 授权 => 进入后台 =>
      Home (主页)：有关注,有收藏夹,all grids Uscales 笑脸 这一排代表着筛选
      Community showcase：热度作品,可以参考关键词和指令
      Personal Feed：关注的人生成的作品
    • 订阅
      manage Sub 可以看到订阅会员,可以按年订阅,也可以按月订阅,如果是长期做图不要选按次200张的
    • 速度模式

一、简介

  • Typora : 官网，支持多平台 https://www.typora.io/
  • 移动硬盘软件文件夹下有序列号版本的Typora
  • Typora其实就是一个浏览器，能解析html + css 语法的 浏览器

二、偏好设置

  • 外观 => 可以获取更多主题，使用Vue主题，下载好的主题放在主题文件夹下，css + 主题文件夹
  • markdown => 严格模式 => 扩展语法勾选12345 => 代码块 => 全部勾选
  • 编辑器 => 全部勾选
  • 图像 => 全部勾选，除了插入时自动转义图片URL，并选择 => 复制图片到./${filename}.assets文件夹
  • 通用 => 启动选项 => 选重新打开上一次使用的文件和目录
  • 通用 => 勾选开发者模式，在页面也可以右键，审查元素，并且知道css的样式属性名，并打开css修改样式

五、源码形式

  • 标题（ 第一种：前面带# ，标题几级，前面就带几个 # ）：
    1. #一级标题
    2. ##二级标题
    3. ###三级标题
    4. ####四级标题
    5. #####五级标题
    6. ######六级标题
    7. #######七级标题
  • 标题（ 第二种：只能表示一级和二级标题， = 和 - 的数量不限制，大于一个即可 ）：
    1. 一级标题
    2. ============
    3. 二级标题
    4. -------------------
  • 标题（ 第三种：和第一种方式不同处是，添加了结束标签，类似 <h1></h1> ）：
    1. #一级标题#
    2. ##二级标题##
    3. ###三级标题 ###
  • *斜体*
  • **加粗**
  • ***斜体加粗***
  • ~~删除线~~
  • cmd+u 下划线
  • --- *** _ _ _ 分割线
  • ==高亮==
  • ~下标~
  • ^上标^
  • ：smile +单词表情 / 输入法自选 / Mac: control+command+space点选,Window:使用 Win键+. 或者Win键+. 点选
  • 列表（无序列表）：
    1. + 1
    2. - 1
    3. * 1
  • 列表（有序列表）：
    1. 1.
    2. 2.
    3. 3.
  • 引用： > 这是一段引用代码
  • 分割线： ---- ***
  • 链接：
    1. [ 这里写网站名 ] {这里写 src}
    2. [ 锚点链接，在src处加上# ] {#匹配名字}
  • 图片： ！[ 图片名称 ] ｛图片地址｝
  • 代码段
    1. 单行使用 ` 内容 ` [ 反引号，键盘中esc下面那个键/英文状态下 ] ` `
    2. 多行使用 ``` 内容 ``` 注意：这里是是前后三个 ``` 反引号
    3. ` ` ` $ sudo apt-get install vim-gnome ` ` `
  • 文本样式：
    1. *字体倾斜* _字体倾斜_
    2. **字体加粗** __字体加粗__
    3. --字体删除--
  • 表格（ ：）是对齐方向
    1. | 项目 | 价格 | 数量 |
    2. | -------- | -----: | :----: |
    3. | 计算机 | \$1600| 5 |

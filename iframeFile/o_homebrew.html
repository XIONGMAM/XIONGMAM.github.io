一、homebrew是什么

Homebrew是Mac的OS X上的软件安装管理工具，类似于apt-get或者yum。拥有安装、卸载、更新、查看、搜索等很多实用的功能。简单的一条指令，就可以实现包管理，而不用你关心各种依赖和文件路径的情况，十分方便快捷。Homebrew Cask是Homebrew的扩展，可以用来安装一些应用。

二、homebrew mac安装 （终端下复制代码，执行，选清华路线）

  • 百度输入 Homebrew CN
  • 看到 homebrew 国内安装脚本 就点进去
  • 选择自己要的脚本 命令
     /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
  • 打开终端，复制粘贴，选择清华大学，，Y，然后等待下载，输入开机指令，等待完成
  • 如果有，则按提示在终端输入：spure /Users/xiongbab/.zprofile
  • 打开终端，输入 brew install git 进行下载安装，这样git 就装好了
  • Git 原下载地址（速度慢）： https://git-scm.com/download/mac
  • Git 国内镜像地址（速度快）： https://www.newbe.pro/Mirrors/Mirrors-Git-For-MacOS/

四、homebrew 基本使用

  • 命令描述
    brew install安装任意软件包 示例：安装wget $ brew install wget
    brew uninstall卸载任意软件包 示例：卸载 git $ brew uninstall git
    brew search查询可用软件包 示例：查找 git $ brew search git 这样就会挪列出软件安装包
    brew info查询任意包信息
    $ brew list查看已安装包的列表
    $ brew upda更新Homebrew
    $ brew -v查看Homebrew版本
    $ brew -hHomebrew帮助信息
    $ brew cask
    $ brew cask search name搜索软件
    $ brew cask install name安装软件
    $ brew cask uninstall name卸载软件
    $ brew update更新软件
    $ brew outdated看下哪些软件可以升级
    $ brew upgrode <xxx>指定软件升级
    $ brew upgrode全部升级
    $ cd "$(brew --repo)" && git remote -v查看 brew.git 当前源
    $ cd "$(brew --repo homebrew/core)" && git remote -v查看 homebrew-core.git 当前源
  • 如果输出是下面这些，就代表不是本地镜像源，可以换成本地镜像源，这样下载会更快点
  • 查看 brew.git 当前源输出
    1. origin https://github.com/Homebrew/brew.git (fetch)
    2. origin https://github.com/Homebrew/brew.git (push)
  • 查看 homebrew-core.git 当前源输出
    1. origin https://github.com/Homebrew/homebrew-core.git (fetch)
    2. origin https://github.com/Homebrew/homebrew-core.git (push)

五、切换 Homebrew 镜像源为中科大镜像源

  • Text 
      替换brew.git:
      cd "$(brew --repo)" && git remote set-url origin https://mirrors.ustc.edu.cn/brew.git

  替换homebrew-core.git:
      cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core" && git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git

  替换homebrew-cask.git:                  
      cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core" && git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git 

  zsh 替换homebrew-bottles镜像，Mac OS在10.15系统开始，默认的shell都换成了zsh   
      echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.zshrc

  修改使其立即生效
      source ~/.zshrc

  bash 替换homebrew-bottles镜像
      echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile

  修改使其立即生效  
      source ~/.bash_profile

  刷新源
      brew update

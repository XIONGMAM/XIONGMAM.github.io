五、列偏移

  • col-md-offset-* ： *代表要偏移的份数，是往左偏移
  • 代码：
      <div> class="col-lg-4 col-lg-offset-4">列偏移</div>

六、列排序

  • 就是把左边的盒子位置替换到右边盒子的位置，实现位置交换的作用
  • 代码：
      <!-- 向右推 -->
  <div class="col-lg-4 col-lg-push-8">列偏移</div>
  <!-- 向左拉 -->
  <div class="col-lg-8 col-lg-pull-4">列偏移</div>

七、列嵌套

  • row里面再套row
  • 代码：
      <div class="container">
      <div class="row">
          <div class="col-lg-4">
              <div class="row">
                  <div> class="col-lg-4"></div>
              </div>
          </div>
          <div> class="col-lg-4 col-lg-pull-1">列偏移</div>
      </div>
  </div>

八、md使用的屏率最高，可以替代lg sm


九、常用样式

  • 排版
    1. 标题
      对h1~h6的标题效果进行覆盖，提供劳务对应的类名，为非标题元素设置样式
      副标题 small标签 或 .small类名
    2. 段落
    3. 强调: 定义了一套类名，强调类名，强调类都是通过颜色来表示强调
      1. .text-muted：提示，使用浅灰色
      2. .text-primary：主要，使用蓝色
      3. .text-success：成功，使用浅绿色
      4. .text-info：通知信息，使用浅蓝色
      5. .text-warning：警告，使用黄色
      6. .text-danger：危险，使用褐色
    4. 对齐:通过定义四个类名来控制文本的对齐风格
      1. .text-left：左对齐
      2. .text-center：居中对齐
      3. .text-right：右对齐
      4. .text-justify：两端对齐

十、设置布局容易

  • Css 
      @media screen and(max-width:767px) {.container {width: 100%;}}
  @media screen and(min-width:768px) {.container {width: 750px;}}
  @media screen and(min-width:992px) {.container { width: 970px;}}
  @media screen and(min-width:1200px) {.container {width: 1170px;}}

十一、注意，bootstrap设置了左右15边距，可以用padding 或 margin 相反负值 或 0消除


十二、栅格系统，页面划分12列 xs-extra small sm-small md-medium lg-large

  • 小于12 会留空白，大于12会自动换行
  • 列嵌套，盒子里套盒子，也会划分12等份
  • 原理是采用百分比%，浮动float来实现一行显示，如果想要间距，使用margin的话，宽度不够盒子会掉下来
    1. 解决办法：外层盒子给padding，内层再多一个盒子100%
    2. 如果要用列嵌套，最好多加一个列row，这样可以取消padding，而且高度会和父级一样
  • 当设计稿与container的1200px不一样的时候，可以修改container宽度来解决
  • 字体图标居中用 element : beforte{ vertical-align: middle;}
